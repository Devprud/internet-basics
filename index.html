World Wide Web (WWW)

Definition:
The WWW is a set of programs, standards and protocols governing the way in which multimedia files are created and displayed on the Internet.
  • Proposed by Tim Berners Lee

  • A large part of the Internet is WWW.

    • USENET

    • Email and Instant Messaging
  • Internet is a collection of computers but WWW is collection of documents linked via special links called hyperlinks.

    encycle.msn.com/getinfo/styles.asp -- > Path of the file

    Protocols

    A protocol is a system of rules that defines how something is to be done.
  • It is a standardised method of transmitting data and/or establishing communications between different devices.
    • HTTP - Hyper Text Tranfer Protocol
      It is an access method used on Internet.
      It is the primary access method for interacting with Internet.
    • HTTPS - Hyper Text Tranfer Protocol Secure
      It ensures the security of the text/information being transferred over the web.
  • Here, the text/information to be transferred is converted into some code words using a technology called SSL certificate
    • SSL - Secure Sockets Layer
    It helps create a secure and encrypted connection between the server and the broswer.
    It ensures that the information is safe from hackers.
